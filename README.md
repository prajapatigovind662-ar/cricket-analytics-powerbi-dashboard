# 🏏 Cricket Analytics Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **Cricket Analytics Dashboard** built using Power BI.
It provides insights into player performance across **Batting, Bowling, and Fielding**.

The dashboard enables users to explore key metrics, compare players, and analyze performance trends effectively.

---

## 🎯 Key Features

* 📊 **Batting Analysis**

  * Total Runs, Batting Average, Strike Rate
  * Performance comparison across players

* 🎯 **Bowling Analysis**

  * Wickets, Economy Rate, Bowling Average
  * Bowling performance insights

* 🧤 **Fielding Analysis**

  * Catches, Run-outs, Stumpings
  * Fielding contribution metrics

* 🔍 Interactive filters and slicers for dynamic analysis
